# 🚀 Virtual_Keyboard

## 🏁 Quick start

``` bash
# Download repository:
git clone ``` link ```

# Install dependencies:
npm install

# Server with live reload at http://localhost:3000/
npm run start
```

| npm              | actions                |
|------------------|------------------------|
| `npm install`    | install                |
| `npm run start`  | development dev-server |
| `npm run build`  | production             |
| `npm run deploy` | deploy gh-pages        |
| `npm run dev`    | development            |
